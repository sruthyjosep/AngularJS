Used OMDB api to query movies and display results.
Single page app is implemented using Angular JS. Have used the concept of views and controllers. Made use of $http service in angular to make the api call
Performed npm install and installed http server, to host it locally
run the command http-server <project-folder-location> in command prompt to host it

Key features of the app

Able to find any movie, tv series or shows based on year and displays relevant information.
